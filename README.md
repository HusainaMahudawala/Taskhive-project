TaskHive is a task management web application built using React.
It allows users to manage tasks with drag and drop functionality.

## Features
- User authentication (Login / Signup)
- Drag and Drop task management
- Task priority (High / Medium / Low)
- User-specific task storage
- Confirmation before moving tasks

## Tech Stack
- React
- Vite
- CSS
- LocalStorage
